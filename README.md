####应用说明：
NV21、RGB 格式转换工具

####版本记录：
v1.1.0: 1.实现动态选择图库的jpg转换成nv21格式文件；
        2.选择转换后的nv21文件进一步转换成rgb格式，并提取bitmap进行显示；
        3.adb连接机台后执行debug目录下的 pull.bat 即可拉取出转换后保存.nv21和.rgb的文件目录；
